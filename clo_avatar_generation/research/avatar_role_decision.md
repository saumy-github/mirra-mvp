# Avatar Role Decision

## Purpose

This document defines the three possible roles for the CLO-native avatar path after comparison work is complete.

## Possible outcomes

### 1. Full simulation and visible avatar

Use this outcome when:

- arrangement reliability is strong
- panel placement is strong
- simulation quality is strong
- body fidelity is good enough for user-facing use

Meaning:

- the CLO-native avatar can become both the simulation body and the avatar shown to the user

### 2. Simulation proxy only

Use this outcome when:

- cloth handling is clearly better on the CLO-native avatar
- but body fidelity is not strong enough for it to be the final visible twin

Meaning:

- CLO-native avatar is used behind the scenes for cloth simulation
- another body representation can still remain the user-facing twin

### 3. Reject CLO-native path

Use this outcome when:

- arrangement or placement gains are weak
- simulation gains are weak
- or total engineering cost is too high relative to the value

Meaning:

- this experimental lane should not become the long-term path
